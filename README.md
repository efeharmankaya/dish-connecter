## Install

### Frontend

Navigate to `/dish-connecter/app`
```
> npm install
```
### Backend

Navigate to `/dish-connecter/api`
```
> pip install -r requirements.txt
```

## Usage

### Frontend
Navigate to `/dish-connecter/app`
```
> npm run start
```

### Backend
Navigate to `/dish-connecter/api`
```
> python3 api.py
```
